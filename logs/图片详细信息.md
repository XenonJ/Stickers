# 主界面

## 图片详细信息

+ 请求方式：GET
+ url：http://127.0.0.1:8000/posts
+ 请求参数：

| 参数           | 是否必须 | 数据类型 | 说明               |
| -------------- | -------- | -------- | ------------------ |
| post_id        | 是       | int   | 帖子id             |


+ 返回参数

```json
{
    'post_time': 2021/3/23 12:30,
    'picture_url':zzh,
    'post_user_id':"http://127.0.0.1:8000/images/0000000.jpg",
    'if_annoyous':no,
    'background_url':"http://127.0.0.1:8000/backgrounds/0000000.jpg",
    'like_number':22,
    'comment_number':11
}
```

```json
{
    'comment_id':23,
    'comment_time':2021/3/23 12:30,

}
```
