# 用户界面

## 用户详细信息

+ 请求方式：GET
+ url：http://127.0.0.1:8000/users/detail/
+ 请求参数：

| 参数           | 是否必须 | 数据类型 | 说明               |
| -------------- | -------- | -------- | ------------------ |
| user_id        | 是       | string   | 用户id             |

+ 返回参数

```json
{
    'user_id': 111,
    'user_name':zzh,
    'profile':"http://127.0.0.1:8000/images/0000000.jpg",
}
{
    'post_id':111,
    'like_time': 1.1.1,
    'picture_url':"http://127.0.0.1:8000/images/0000000.jpg",
}
{
    'post_id':111,
    'comment_time': 1.1.1,
    'picture_url':"http://127.0.0.1:8000/images/0000000.jpg",
    'comment':评论内容
}
```
