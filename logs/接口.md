# 登陆注册接口

在登录后返回一个token作为用户的唯一认证id，存在cookie中。

之后其他接口中所有需要用户id的参数统一用token代替。

## 注册

+ 请求方式：POST
+ url：http://127.0.0.1:8000/users/register/
+ 请求参数：

| 参数           | 是否必须 | 数据类型 | 说明               |
| -------------- | -------- | -------- | ------------------ |
| user_id        | 是       | string   | 用户id             |
| username       | 是       | string   | 用户用于展示的昵称 |
| password       | 是       | string   | 用户密码           |
| student_number | 否       | int      | 用户实名认证的学号 |
| profile        | 是       | file     | 用户头像           |

+ 返回参数

```json
{
    "status": 200/404,
    "error": 错误信息,
}
```

## 登录

+ 请求方式：POST
+ url：http://127.0.0.1:8000/users/login/
+ 请求参数：

| 参数     | 是否必须 | 数据类型 | 说明     |
| -------- | -------- | -------- | -------- |
| user_id  | 是       | string   | 用户id   |
| password | 是       | string   | 用户密码 |

+ 返回参数

```json
{
    "data":
    {
        "token": 返回的用户登录验证信息，保存在cookie中（相当于userid）,
    	"user permissions": 用户权限
    },
    "status": 200/404,
    "error": 错误信息,
}
```

```json
        }
```

```json
}
```

## 帖子详细信息

+ 请求方式：GET
+ url：http://127.0.0.1:8000/pages/post_detail/
+ 请求参数：

| 参数    | 是否必须 | 数据类型 | 说明                |
| ------- | -------- | -------- | ------------------- |
| token   | 是       | string   | 用户id（用户token） |
| post_id | 是       | string   | 帖子id              |

+ 返回参数：

```json
{
    "data":
    {
        "post":
        {
            "if_self": 是否为浏览者本人发的贴,
            "post_time": 发帖时间,
            "picture_url": 帖子url,
            "background_url": 背景url,
            "like_num": 点赞数,
            "comment_num": 评论数,
            "if_anonymous": 是否匿名，如果为true，则以下的key对应值全为空,
            "user_id": 发帖人id,
            "user_name": 发帖人昵称,
            "profile_url": 发帖人头像url,
        },
        "comments":
        [
            {
                "if_self": 是否为浏览者本人发的评论,
                "comment_id": 评论id,
                "comment_time": 评论时间,
                "comment": 评论内容,
                "like_num": 点赞数,
                "if_anonymous": 是否匿名，如果为true，则以下的key对应值全为空,
            	"user_id": 评论者id,
            	"user_name": 评论者昵称,
            	"profile_url": 评论者头像url,
            },
            {
                
            }
        ]
    },
    "status": 200/404,
    "error": 错误信息,
}
```

# 用户操作接口

## 上传帖子

+ 请求方式：POST
+ url：http://127.0.0.1:8000/users/upload_post/
+ 请求参数：

| 参数                 | 是否必须 | 数据类型 | 说明           |
| -------------------- | -------- | -------- | -------------- |
| token                | 是       | string   | 用户token      |
| x_coordinate         | 是       | int      | 帖子x坐标      |
| y_coordinate         | 是       | int      | 帖子y坐标      |
| rotation_angle       | 是       | int      | 帖子旋转角度   |
| picture              | 是       | file     | 帖子内容       |
| background_selection | 是       | int      | 选择的背景图片 |
| if_anonymous         | 是       | bool     | 是否匿名       |

+ 返回参数：

```json
{
    "status": 200/404,
    "error": 错误信息,
}
```
