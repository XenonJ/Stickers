# 主界面

## 显示信息

+ 请求方式：GET
+ url：http://127.0.0.1:8000/posts
+ 请求参数：

| 参数           | 是否必须 | 数据类型 | 说明               |
| -------------- | -------- | -------- | ------------------ |
| post_id        | 是       | int   | 帖子id             |
| picture_url        | 是       | string   | 图片url             |
| Rotation_angle        | 是       | int   | 旋转角度             |
| x_coordinate        | 是       | int   | 界面位置x轴            |
| y_coordinate        | 是       | int   | 界面位置y轴            |

+ 返回参数

```json
{
    'status': 200/404,
    'error': 错误信息,
}
```
